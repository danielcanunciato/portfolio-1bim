## Relatório sobre a eficácia dos editores de texto
Procurar um melhor entendimento para vocês sobre edição de código, venho apresentar uma boa alternativa à vocês.

Inicialmente, temos que começar com algo mais simples para um entendimento mais eficaz sobre os editores de texto, então iremos apresentar primeiro a formatação e visualização, sobre isso iremos utilizar VSCode (Visual Studio Code) como um exemplo.


**Formatação e Visualização**

A formatação e a visualização se tratam de elementos cruciais para a edição sobre qualquer tipo de código, em qualquer tipo de editor de texto básico que você encontre, seja na navegação na internet ou em outras plataformas, os códigos que você escreve se parecerá com um texto qualquer, sem elementos de destaque nas palavras do seu código, sem indicações visuais.


**Autocompletar**

Os editores de texto não possui uma ferramenta principal onde nosso VSCode possui, que é o autocompletar das sintaxes dos códigos, onde isso ajuda você a ter um entendimento melhor sobre o código e o que você pode fazer de melhora em sua construção. 


**Verificações de Erros**

As verificações de erro é algo essencial para o seu código, pois são os que demonstram visualmente seus erros para poderem serem corrigidos antes de qualquer teste, editores de textos básicos não possuem essa ferramenta onde pode se tornar um problema.


**Organização e Implementação**

Se você for fazer algum projeto ambicioso ou grande em geral que exija uma organização mais profissional, o VSCode é perfeito nesses requerimentos, sendo possível adição de comentários para uma melhor compreensão do código e organizações.

Já na questão de implementação você poderá encontrar uma vasta quantidade de bibliotecas com uma grande quantidade de extensões que poderá utilizar para facilitar a construção dos seus códigos, como por exemplo instalar a linguagem Python diretamente no VSCode ou uma extensão para conseguir ver se seu código está funcionando na web enquanto você programa ele.
